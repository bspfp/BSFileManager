	Alt + (Insert, Delete, PageUp, PageDown)를 가지고 현재 선택된 뷰의 경로를 다른 뷰로 복제하는 기능 추가
	주소 표시줄에서 데이터 입력하다 ESC를 누르거나 포커스를 잃으면 원래 내용으로 복구
	클립보드 관련 자료

	파일 경로 복사 기능
	찾기 기능
		기능
			경로복사
				선택된 파일을 텍스트로 클립보드에 경로 복사
			삭제
				선택된 파일을 휴지통으로 이동
				휴지통 이동 불가능시에는 물어보고 지우기
			복사
				선택된 파일을 쉘에 붙여 넣을 수 있게 복사하기
			찾은 결과에서 헤더로 정렬하기
	필터
		검색과 같이 일반 모드, 정규식 모드를 선택
		모달 다이얼로그로 필터 입력
		필터링 된 뷰는 테두리 색깔을 파랑에서 빨강으로 변경
	복수의 파일 이름 변경
		선택된 파일의 이름을 변경
		태그를 사용해서
			태그 종류
				파일 이름(확장자 제외)
				파일 확장자
				상위 폴더 이름
				현재 시간
				자동 넘버링
					자리수 입력
						3자리면 001, 002 이렇게
					1부터 시작
		중복된 이름이 될 경우에는 파일명 뒤에 넘버링 자동으로
			가나다_1.txt, 가나다_2.txt
			중복된 수를 가지고 자리수 결정
